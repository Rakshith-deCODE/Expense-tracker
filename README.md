# Expense-tracker
Expense Tracker A CLI-based Python application to record, view, and manage daily expenses. It stores data locally and calculates total spending, helping users track their finances in a simple and efficient way.

## 📌 Description
A simple CLI-based Python application to track daily expenses and calculate total spending.

## 🚀 Features
- Add expenses  
- View all expenses  
- Calculate total  

## ▶️ How to Run
python main.py
